## Requirements
- Neovim
- Git
- LazyGit
- LazyVim
- ripgrep
- fd
- fzf
## Terminal and Shell
- iTerm2
- bash
- zsh
- fish
- nu
- Neovide (UI Integrated with Neovim)
## Installations
```zsh
cd ~/.config/
git clone --recursive https://github.com/soullessness1611/nvim-public.git && \
	mv ~/.config/nvim-public ~/.config/nvim

nvim
```
