## Requirements
- Neovim >= 0.9.0
- Git >= 2.29.0
- LazyGit
- LazyVim
- ripgrep
- fd
- fzf

- zsh

- nu

## Installations
```zsh
git clone --recursive https://github.com/soullessness1611/nvim-public.git ~/.config/ && \
	mv ~/.config/nvim-public ~/.config/nvim
```
